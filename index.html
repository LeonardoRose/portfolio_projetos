Projeto em Ciências de Dados

Previsão de vendas na rede de farmácias Rossmann

O objetivo desse projeto foi realizar a previsão de vendas das próximas 6 semanas de cada loja da rede Rossmann usando Machine Learning.

O projeto seguiu as etapas do CRISP-DM e foi desenvolvido em Python e o resultado final foi hosteado na cloud permitindo assim que previsões de cada loja possam ser consultadas via Telegram.

Para avaliação do melhor modelo foi usado o erro MAPE (Erro Percentual Médio).
