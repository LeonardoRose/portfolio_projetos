    Projeto em Ciências de Dados

    Previsão de vendas em uma rede de farmácias com Regressão

    O CEO da rede Rossmann necessita uma maior acuracidade na previsão de vendas para que consiga tomar decisões mais assertivas.

    Nesse projeto, o objetivo é realizar a previsão de vendas das próximas 6 semanas de cada loja da rede Rossmann usando Machine Learning.

    O projeto seguiu as etapas do CRISP-DM, foi desenvolvido em Python e o resultado final foi hosteado na cloud permitindo assim que previsões de cada loja possam ser consultadas via Telegram.

    Para avaliação do melhor modelo foi usado o erro MAPE (Erro Percentual Médio).

    Também foi utilizado:

    • Jupyter Notebook
    • Git e Github
    • Telegram API
    • Heroku Cloud

    Propensão de compras com Classificação

    A seguradora que está com apenas 40% da força operacional, está lançando um novo produto e precisa saber quais os clientes mais propensos a aderir ao novo produto.

    Pela inviabilidade de contactar 100% da base, o objetivo desse projeto é, usando Machine Learning, atribuir a cada cliente da base a sua respectiva propensão de compra para que seja oferecido o novo produto aos 40% de clientes mais propensos.

    O projeto seguiu as etapas do CRISP-DM, foi desenvolvido em Python usando os conceitos de Learn to Rank e a propensão de compra de cada cliente pode ser consultados no Google Sheets.

    Também foi utilizado:

    Competições

    1º Lugar na 7ª edição do Hackday da ComunidadeDS

    Como capitão da equipe, desenvolvemos durante 2 dias um modelo de Regressão Linear para realizar a previsão de vendas das próximas 5 semanas da rede ficticia EletroPlaze Store.

    A métrica de avaliação foi o RMSE, onde nossa equipe atingiu a primeira colocação, com o menor RMSE: 19.347,90. O RMSE baseline era de: 93.028,13

    Ao aderir nosso modelo, a rede EletroPlaza conseguiria um lucro semanal incremental de 54%, equivalente a 400 mil dólares.

    A solução construída:

    • Seguiu as etapas do CRISP-DM.
    • O algoritmo que melhor desempenhou foi CatBoost Regressor.
    • Usou KNN para substituição de valores nulos em uma das colunas.
    • Código do modelo vencedor
